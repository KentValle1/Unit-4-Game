Crystal Collector

Here is what you need to know to play this game!


  • The game will start with a random number.
  • You will need to use the four Crystals bellow in order to match that number.
  • The crystals have a set number associated with them.
  • You will need to click the crystals to add up your number to the nummber given to you.
  • If you go over the number it will count as a loss, so click very carefully.
  • Otherwise, you will get a win for correctly matching the number.
